A professional, educational typing tutor interface with a playful "Claymorphism" aesthetic.

DESIGN SYSTEM (REQUIRED):

Platform: Web, Desktop-first

Theme: Claymorphism (Soft 3D, chunky, bubbly, playful)

Background: Soft Lavender-Blue (#EEF2FF)

Primary Accent: Indigo (#4F46E5) for active elements

CTA: Energetic Orange (#F97316) for "Start" and "Next" actions

Text Primary: Deep Navy (#1E1B4B)

Shapes: Generously rounded corners (24px), thick borders (3px), double shadows for a "toy-like" 3D depth.

Page Structure:

Header: "Array30 Typing Tutor" in a bold font.

Metrics Row: Three elevated cards for "Speed (WPM)", "Accuracy", and "Progress".

Typing Container: A large central card with a "clay" border. Inside, the target text is displayed in a monospaced font. The current character is highlighted with an Indigo pulse.

Action Footer: "Restart" and "New Lesson" buttons with a chunky, pressable look.
